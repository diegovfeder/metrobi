# Metrobi Interview Challenge

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To run the React challenge:

```bash
bun run react-dev
```

This project was created using `bun init` in bun v1.1.18. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
